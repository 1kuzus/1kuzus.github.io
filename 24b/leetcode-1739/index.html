LeetCode 1739.放置盒子

null likes · null views · 2024-05-29

原题链接：放置盒子

数学找规律题目，我推出了O(1)的函数f(i)，表示最下层有i个盒子时，最大的可能总盒子数。然而题目是想反求，因此可以使用二分。

二分的复杂度为log级别，有更优雅的整体复杂度为常数的数学解法，可以参考官方题解。

不过，我们可以考虑使用不精确的数学方法优化二分的初始区间，考虑对f(i)进行放缩。

f(i)>62i⋅28i​​​=32​​i1.5

推导过程就省略了，然后反解得到：

i<(2​3​f(i))2/3<(2.13⋅f(i))2/3

这是对区间右端的近似。同样对于区间左端，可以在n足够大时保证：

i>(2⋅f(i))2/3

这里对常数的选取是较为随意的。经过这个优化可以把二分区间压缩到非常小的范围。

C++
class Solution {
public:
    int f(int i)
    {
        int a=-0.5+sqrt(1+8*i)/2;
        int b=i-a*(a+1)/2+1;
        int c=b*(b+1)/2;
        return (long long)a*(a+1)*(a+2)/6+c;
    }
    int minimumBoxes(int n) {
        //int l=0,r=pow(2.13*n,2.0/3);
        int l=n<500?0:pow(2*n,2.0/3),r=pow(2.13*n,2.0/3);//更紧的约束
        while(l+1<r)
        {
            int mid=l+(r-l)/2;
            if(f(mid-1)>=n) r=mid;
            else l=mid;
        }
        return r;
    }
};
